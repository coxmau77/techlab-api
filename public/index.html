Bienvenido a TechLab

Esta es una aplicación de demostración para gestionar un inventario de productos tecnológicos.

Puedes registrarte para obtener una cuenta, iniciar sesión y comenzar a añadir, editar y eliminar tus propios productos.

El backend está construido con Node.js, Express y Firebase. El frontend es HTML, CSS y JavaScript puro.
